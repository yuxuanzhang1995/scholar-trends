# Scholar Trends

Landing page for **Scholar Trends** — ten MCP tools that turn arXiv + OpenAlex into
research-trend analytics and a personal research copilot for Claude. Extends
[arxiv-mcp-server](https://github.com/blazickjp/arxiv-mcp-server).

**Live:** https://yuxuanzhang1995.github.io/scholar-trends/

A single static page (`index.html`, charts via Chart.js) deployed with GitHub Pages.
All figures come from real OpenAlex / arXiv calls.
