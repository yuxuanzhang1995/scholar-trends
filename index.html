the toolbelt

An analytics engine, plus a personal-copilot layer on top.

keyword_trends — a topic's rise/fade on arXiv
combo_impact — which topic mixes are high-impact
segment_focus — who works on it: country / continent / institution
venue_trends — a topic's output per journal (OA-filterable)
lead_lag — does arXiv lead the journals?
taste_profile — what a journal over/under-emphasizes
venue_info — tier + real JCR impact factor + OpenAlex stats
length_trends — are papers getting longer over time?
research_brief — name/keywords → reading list + open problems
open_problems — the open questions in a field

how it works

arXiv + OpenAlex

Preprints from arXiv plus 250M+ works with venue, institution, topic and citation metadata from OpenAlex. Both free.

No API key

Runs in the OpenAlex polite pool with just an email — nothing to sign up for.

Two layers

An analytics engine (trends, impact, venue taste) with a personal copilot (reading lists, open problems) on top.

Injection-aware

Paper text is treated as untrusted external input — it never drives actions.

install

Add to your Claude MCP config, then ask Claude to use the tools:

{
  "mcpServers": {
    "scholar-trends": {
      "command": "uvx",
      "args": ["arxiv-mcp-server"],
      "env": { "OPENALEX_MAILTO": "you@example.com" }
    }
  }
}
